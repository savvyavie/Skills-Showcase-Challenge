# Description
This is a submission by Lance Xavier Castro for the Skills Showcase Challenge for the internship application to Mata Technologies.

# Technologies Used
This portfolio web page was built with **HTML, CSS and JavaScript**.

# Demo Instructions
Download the files and place them all in one folder, and open index.html to view the page. Alternatively, go to [this site](skills-showcase-challenge.vercel.app) to view it online.

# Features
## Required
- **Home** - Name, profile picture (header icon), bio
- **Projects** - Titles, descriptions, tech used, images and links
- **Contact** - Simulated contact form with toast notification 
- **Responsive** — Mobile mode engages based on screen/window size
- **Smooth navigation** — Nav bar links scroll to section when clicked
## Optional
- **Dark/Light mode toggle** - persists via `localStorage`
- **Animations** — Scroll fade-ins, hover effects, active text box effects, etc.
- **Deployment** - Deployed on Vercel
